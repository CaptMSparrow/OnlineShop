Administrator

Up 
get /products
Просмотр всех товаров (productsGet)
Получение списка всех товаров в магазине с поддержкой пагинации и сортировки.

Query parameters

page (optional)
Query Parameter — Номер страницы для пагинации. default: 1
limit (optional)
Query Parameter — Количество товаров на странице. default: 10
sort_by (optional)
Query Parameter — Поле для сортировки (price, name). default: name
order (optional)
Query Parameter — Порядок сортировки. default: desc

Return type

array[product]

Example data

Content-Type: application/json
[ {
  "id_product" : 123,
  "price" : 100,
  "name" : "Кольцо Змея",
  "description" : "Серебряное кольцо в виде змеи, безразмерное",
  "category" : "Кольца"
}, {
  "id_product" : 123,
  "price" : 100,
  "name" : "Кольцо Змея",
  "description" : "Серебряное кольцо в виде змеи, безразмерное",
  "category" : "Кольца"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Список товаров
